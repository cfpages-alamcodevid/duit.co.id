# Standar Laptop Modal Usaha 2026

Dokumen ini dipakai sebagai acuan saat artikel atau research menyebut laptop sebagai modal usaha. Tujuannya sederhana: jangan meremehkan laptop sebagai alat produksi.

## Prinsip Utama

- Laptop boleh dianggap Rp 0 hanya kalau pembaca memang sudah punya laptop yang stabil untuk beban kerja tersebut.
- Jangan rekomendasikan laptop Rp 1-2 juta untuk pekerjaan produktif digital, kecuali konteksnya benar-benar darurat dan hanya untuk input ringan.
- Jangan menulis "i7" tanpa generasi. i7 generasi tua bisa lebih buruk dari i5 generasi baru.
- Untuk 2026, RAM 16GB adalah baseline aman untuk kerja digital serius karena browser, OS, aplikasi meeting, storage sync, dan tool kerja makin berat.
- Hindari HDD. Minimal SSD 256GB, dan 512GB untuk kerja dengan file besar, desain, video, coding, atau banyak dokumen klien.

## Baseline Spek

| Beban kerja | Spek minimal realistis 2026 | Budget bekas realistis |
|---|---:|---:|
| Admin, dokumen, spreadsheet, marketplace, customer support | Intel Core i5 Gen 10 / Ryzen 5 4000, RAM 16GB, SSD 256GB | Rp 4-6,5 juta |
| Coding ringan, web/no-code, dashboard, banyak tab browser | Intel Core i5/i7 Gen 10-11 / Ryzen 5 4000-5000, RAM 16GB, SSD 512GB | Rp 5-8 juta |
| Desain grafis ringan, Canva/Figma, PPT premium | Intel Core i5/i7 Gen 10-11 / Ryzen 5 4000-5000, RAM 16GB, SSD 512GB, layar IPS/FHD | Rp 5-9 juta |
| Video editing 1080p, animasi, CAD, GIS, 3D ringan | Intel Core i5/i7 Gen 10-11 H-series / Ryzen 5-7 4000-5000, RAM 16GB, SSD 512GB, GPU GTX 1650/RTX 3050 bila rendering berat | Rp 8-14 juta |

## Seri Bekas yang Masuk Akal

- Lenovo ThinkPad T14 Gen 1/Gen 2, ThinkPad T490, ThinkPad T480 hanya sebagai fallback murah jika RAM dan SSD sudah di-upgrade.
- Dell Latitude 5410/5420/5430.
- HP EliteBook 840 G7/G8.
- Untuk CAD, GIS, animasi, dan 3D: cari workstation/gaming bekas yang pendinginannya jelas, bukan ultrabook tipis yang mudah throttling.

## Red Flags Saat Beli Bekas

- CPU Intel Gen 6/7 atau lebih tua untuk kerja utama. Masih bisa hidup, tapi riskan untuk Windows 11, security update, dan aplikasi modern.
- RAM 8GB soldered yang tidak bisa di-upgrade.
- Storage masih HDD.
- Baterai drop parah, engsel rusak, keyboard banyak mati, atau layar bergaris.
- "i7 murah" tanpa generasi jelas.
- Laptop bekas gaming yang bekas mining/render berat tanpa dicek suhu dan kondisi kipas.

## Cara Menulis di Artikel

Jika laptop menjadi modal usaha, gunakan formulasi seperti ini:

> Kalau sudah punya laptop yang stabil, jangan beli dulu. Tapi kalau laptop sering hang, RAM masih 4-8GB, atau masih HDD, masukkan upgrade sebagai modal usaha. Untuk 2026, patokan aman minimal Intel Core i5 Gen 10/Ryzen 5 4000, RAM 16GB, SSD 256-512GB. Bekas ex-kantor seperti ThinkPad T14, Dell Latitude 5410/5420, atau HP EliteBook 840 G7/G8 biasanya lebih masuk akal daripada laptop murah baru.

Untuk pekerjaan berat:

> Untuk video, animasi, CAD, GIS, dan 3D, jangan pakai patokan laptop admin. Siapkan minimal RAM 16GB, SSD 512GB, dan GPU dedicated seperti GTX 1650/RTX 3050 jika workflow-nya rendering. Budget bekas realistis lebih dekat ke Rp 8-14 juta.
