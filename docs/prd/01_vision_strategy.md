# Project Vision & Strategy: Duit.co.id

## 1. Vision
Menjadi platform ekosistem finansial nomor #1 di Indonesia yang menyediakan solusi personalisasi berdasarkan tingkatan ekonomi (Tier 0 - Tier 4), pendidikan, domisili, dan usia melalui konten edukasi, alat bantu (tools) praktis, dan jembatan ke layanan profesional.

## 2. Core Strategy
- **Personalization:** Tailoring content and tools to user tiers (Survival to Legacy).
- **Gamification/Discovery:** Using a "Smart Financial Quiz" as the primary entry point.
- **Ecosystem Integration:** Bridging educational content with professional services (Properti.id, Franchise.id).
- **Monetization:** Affiliate hooks, Lead generation pipeline, and Digital products (Academy).

## 3. Development Roadmap
- **Phase 1 (MVP):** Auth (Clerk), Landing Page, Quiz, and initial content/tools for Tier 0 & 1.
- **Phase 2 (Content & Law):** Law Library implementation and YouTube Gate integration.
- **Phase 3 (Scaling):** External ecosystem integration and Academy marketplace launch.
