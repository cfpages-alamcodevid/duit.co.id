Step 3: Set up and run test
- Details: Randomize traffic, run for 1-2 weeks minimum
- Example: 50/50 split, monitor daily
- Warning: Don't stop test too early

Step 4: Analyze results with statistical significance
- Details: Use calculators, check p-value <0.05
