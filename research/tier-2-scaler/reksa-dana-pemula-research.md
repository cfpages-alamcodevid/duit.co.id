# 2. Core Editorial Angle
Mayoritas konten hanya menjelaskan definisi. Nilai tambah artikel ini: kerangka pemilihan reksa dana berbasis tujuan, horizon, risiko, biaya, dan pajak Indonesia.

# 3. Indonesia Policy and Tax Context
- Reksa dana diawasi OJK; pembelian umumnya via APERD/platform berizin.
- Pajak di level investor berbeda per instrumen dasar (misal kupon obligasi dikenakan pajak di level produk/instrumen, detail perlu verifikasi).
- Penting jelaskan perbedaan NAB/UP harian versus return jangka panjang.

Angka acuan:
- Dana darurat ideal 3-12 bulan pengeluaran sebelum investasi agresif.
- Biaya pembelian/penjualan kini banyak 0% di platform tertentu (Perlu verifikasi).

# 4. Example Numbers
Simulasi DCA:
- Investasi rutin Rp1.000.000/bulan selama 5 tahun
- Return asumsi 6%/tahun: nilai akhir sekitar Rp69-70 juta (Perlu verifikasi)
- Return asumsi 10%/tahun: nilai akhir sekitar Rp77-78 juta (Perlu verifikasi)

Perbandingan sederhana:
- Pasar uang: volatilitas rendah, cocok <= 1-2 tahun.
- Pendapatan tetap: moderat, cocok 2-5 tahun.
- Saham: volatilitas tinggi, cocok > 5 tahun.

# 5. Risks and Notes
- Risiko ekspektasi return terlalu tinggi karena lihat performa 1 tahun terakhir.
- Risiko mismatch horizon: dana DP rumah 1 tahun ditempatkan di reksa dana saham.
- Risiko fee tersembunyi jika tidak baca fund fact sheet.

Mitigasi:
- Gunakan checklist: tujuan, tenor, drawdown yang bisa ditoleransi.
- Evaluasi berkala per kuartal, bukan harian.
- Diversifikasi antar kelas aset.

# 6. Persona Scenarios
Persona A: Andi, 27, gaji Rp8 juta, pengeluaran Rp5 juta, target DP nikah 2 tahun.
Strategi: dominan pasar uang + porsi kecil pendapatan tetap.

Persona B: Mira, 31, gaji Rp18 juta, target pensiun dini 20 tahun.
Strategi: kombinasi saham + pendapatan tetap dengan DCA otomatis.

# 7. Myths to Debunk
- "Reksa dana pasti untung." Tidak, nilai bisa naik turun.
- "Harus punya modal besar." Tidak, banyak platform mulai Rp10 ribu-Rp100 ribu.
- "Pantau tiap jam biar cuan." Tidak, ini mendorong overtrading emosional.

# 8. Action-Step Outline
1. Tetapkan tujuan finansial dan tenggat.
2. Pastikan dana darurat dan proteksi dasar selesai.
3. Pilih kelas reksa dana sesuai horizon.
4. Baca fund fact sheet: AUM, biaya, komposisi, risiko.
5. Aktifkan DCA bulanan.
6. Rebalancing setiap 6-12 bulan.
