### 5. Legal & Regulatory Requirements
```
Requirement: Arm's Length Principle (ALP) application in all related-party transactions
Regulation: Article 18 of Income Tax Law (UU PPh), implemented via PMK 172/2023
Process: Compare transaction conditions to independent parties; prepare preliminary assessment for 6 transaction types (services, intangibles, loans, restructuring, etc.)
Cost: No direct cost, but penalties for non-compliance; professional fees Rp 20-100 million/year for compliance
Timeline: Documentation ready by April 30 next year; submit within 1 month if requested
Authority: Directorate General of Taxes (DGT)
Link: https://pajak.go.id/sites/default/files/2025-05/REGULATION%20OF%20THE%20MoF%20No.172%20year%202023%20-%20English.pdf

Requirement: Transfer Pricing Documentation (Master File, Local File, CbCR if applicable)
Regulation: PMK 172/2023 Article 16
Process: Prepare ex-ante (Price-Setting Report) for certain transactions; file statement with annual tax return
Cost: Preparation costs Rp 50-300 million depending on complexity
Timeline: Available by tax return filing date; CbCR within 1 year after fiscal year end
Authority: DGT
Link: https://www2.deloitte.com/content/dam/Deloitte/id/Documents/tax/id-tax-transfer-pricing-alert-jan2024-en.pdf

Requirement: Advance Pricing Agreement (APA) for certainty
Regulation: PMK 172/2023 Articles 55-71
Process: Submit application 2-6 months before transaction; bilateral/multilateral possible
Cost: Application fee and consulting Rp 100-500 million
Timeline: 6-18 months approval; 3-5 year term
Authority: DGT
Link: Same as above
```

### 9. Action Steps Outline
```
Step 1: Identifikasi pihak terkait dan transaksi
- Details: Mapping ownership >25%, control, family relations; list all intercompany flows (goods, services, loans, intangibles)
- Example: Buat matrix entitas grup dengan transaksi >Rp 5B
- Warning: Jangan lewatkan hidden relationships via trusts atau indirect ownership

Step 2: Lakukan preliminary assessment untuk transaksi wajib
- Details: Evaluasi komersial justification, economic benefits, substance check untuk 6 jenis transaksi
- Example: Untuk jasa intra-group, bandingkan dengan biaya eksternal
- Warning: Tanpa ini, transaksi dianggap tidak ALP otomatis

Step 3: Tetapkan harga ex-ante dengan Price-Setting Report
- Details: Analisis fungsi/aset/risiko, pilih metode (CUP preferred), select comparables, justify economically
- Example: Untuk ekspor produk, gunakan CUP dengan harga pasar lokal
- Warning: Retroactive tidak diterima; siapkan sebelum transaksi

Step 4: Siapkan dokumentasi lengkap (Master File, Local File, CbCR)
- Details: Master File global overview, Local File analisis detail, CbCR jika revenue >Rp 11.4T global
- Example: Gunakan template OECD BEPS Action 13
- Warning: Harus dalam bahasa Indonesia, siap dalam 1 bulan jika diminta

Step 5: File laporan pajak dan monitor compliance
- Details: Lampirkan statement kesiapan dokumentasi; siap audit; pertimbangkan APA untuk certainty
- Example: Konsultasi dengan konsultan pajak bersertifikat
- Warning: Delay submission bawa penalty interest uplift 15%
```
