# Research: Cara Hemat Pulsa, Internet, dan Kuota

## Reality Check: Communication Costs in Indonesia 2026

### Average Monthly Spending (UMR Worker)
```
- Main quota: Rp 100.000-150.000
- Call credit: Rp 25.000-50.000
- Electricity token: Rp 100.000-200.000
- E-wallet (grab/food): Rp 150.000-300.000
- Wi-Fi: Rp 200.000-300.000
Total: Rp 575.000-1.000.000 (15-25% of UMR salary!)
```

### Why Quota Disappears Fast
1. **"Unlimited" is a myth**: All operators have FUP (Fair Usage Policy)
2. **Background apps**: WhatsApp, IG, FB run continuously
3. **Auto-play videos**: TikTok, Reels, FB auto-play
4. **Auto-updates**: Play Store/App Store download automatically

### Daily Data Consumption (Standard Activity)
```
WhatsApp (chat + voice notes): 20-50 MB
Maps/Waze (30 min navigation): 15-30 MB
Spotify/YouTube Music: 50-100 MB (offline 0 MB)
Instagram (scroll only): 30-100 MB
TikTok (15 min scroll): 100-300 MB
Zoom/Meet (1 hour): 500 MB-1 GB
Netflix (1 hour HD): 1-3 GB
```

## Operator Selection by Location

### Big Cities (Jakarta, Surabaya, Bandung)
- **Recommendation**: Telkomsel (Loop/Simpati) or XL

### Hybrid (City-Village Commute)
- **Recommendation**: Indosat Ooredoo (IM3)
- **Why**: Decent stability, medium price
- **Price**: Rp 60.000-90.000 for 10GB

## Quota Buying Strategies

### Rule 1: Avoid Daily Quota (Except Emergency)
- Daily 1GB = Rp 5.000/day = Rp 150.000/month for 30GB
- Monthly 15GB package = Rp 80.000 only
**Use daily only for**: Sudden out-of-town events, emergency internet

### Rule 2: Choose "Regular" Not "Social Media" Packages
```
Social media packages:
- WhatsApp only: Rp 30rb
- Instagram only: Rp 25rb
- YouTube only: Rp 40rb
- Regular 10GB: Rp 80-95rb (all apps)
```
**Mistake**: Buy all social packages (Rp 95rb total) but can't access Maps

### Rule 3: Utilize Free Wi-Fi (Be Careful)
**Safe locations**:
- Office/workspace (usually free)
- Indomaret/Alfamart (slow, chat only)
- Big cafes (Starbucks, Kopi Kenangan) - 1-2 hours free
- Malls - food court areas

**Wi-Fi safety rules**:
1. Never access internet banking on public Wi-Fi
2. Turn off file sharing (AirDrop, Nearby Share)
3. Use VPN for sensitive data (free: ProtonVPN, Windscribe)
4. Don't save Wi-Fi passwords (prevent auto-connect)

### Rule 4: Buy Combo Packages (Quota + Calls + SMS)
Look for packages including:
- Large quota
- Call credit (for non-network calls)
- SMS (still used for bank verification)

## HP Settings to Save Quota

### Setting 1: Turn Off Background Data
**Android**: Settings → Apps → [App Name] → Mobile data & Wi-Fi → Turn OFF "Allow background data"
**iPhone**: Settings → General → Background App Refresh → Turn OFF

### Setting 2: Enable Data Saver
**Android**: Settings → Network & internet → Data Saver → ON
**iPhone**: Settings → Cellular → Low Data Mode → ON

### Setting 3: Disable Auto-Updates
**Play Store**: Settings → Network Preferences → "Do not auto-update apps"
**App Store**: Settings → App Store → Turn OFF "App Updates"

### Setting 4: Reduce Streaming Quality
**YouTube**: Settings → Video Quality Preferences → "Data Saver"
**TikTok**: Profile → Settings → Data Saver → ON
**Instagram/Facebook**: Settings → Media → Video Autoplay → "Never"

### Setting 5: Control Tethering/Hotspot
- Turn OFF by default
- Only enable when needed for laptop work
- Turn OFF immediately after use

## Call & SMS Saving Tricks

### Trick 1: Use WhatsApp/Telegram for All Communication
- Replace calls with **Voice Note WhatsApp** (~2-5 MB for 30 min vs Rp 2.000-3.000 call)
- Use **Telegram** for meetings (less data than Zoom)

### Trick 2: Call During "Cheap Hours"
**Telkomsel**: Off-peak 1:00-5:00 AM (Rp 10/sec vs Rp 25/sec)
**Indosat**: Night call package Rp 10rb for unlimited calls 12AM-6AM

### Trick 3: WhatsApp Broadcast Instead of SMS
- Broadcast message (free) vs SMS massal (Rp 50/SMS × 50 people = Rp 2.500)

### Trick 4: VoIP Apps for International Calls
- WhatsApp Call (free via quota)
- Google Voice (if have account)
- Skype (cheap for regular phone numbers)

## Quota Check & Emergency Protocols

### Set Up Notifications
**Telkomsel**: MyTelkomsel → Settings → Enable "Quota Reminder"
**Indosat**: IM3 App → Settings → Enable "Data Usage Alert"
**Universal**: SMS check: Telkomsel "KUOTA" to 3636, Indosat "DATA" to 123

### Emergency When Quota Runs Out
1. **Turn OFF data for all apps except critical ones** (WhatsApp, Maps only)
2. **Use public Wi-Fi in emergency** (Indomaret, cafe)
3. **Buy emergency add-on**: Telkomsel Rp 2.000 = 1GB, Indosat Rp 3.000 = 1GB
4. **Borrow friend's hotspot** (treat them coffee Rp 10.000)

### Danger Signs: Quota Runs Out Too Fast
- Background apps running (check settings)
- Faulty battery causing restart with data ON
- Someone stealing connection (hotspot leaked)
- **Check**: Settings → Data usage → See which app uses most

## Common Fatal Mistakes

1. **Charging overnight while watching YouTube** → 300MB for 1 hour while charging
2. **Leaving Maps open continuously** while riding → Use only at start, turn off screen (GPS still works)
3. **Auto-updates on mobile data** → 500MB lost for TikTok update (300MB)
4. **Watching "time-wasting" videos on data** → 2 hours TikTok = 2GB (Rp 20.000 wasted)
5. **Not turning OFF data while sleeping** → 100-300MB/night for background updates

## Emergency Scenarios & Solutions

### Scenario 1: Urgent Online Meeting, Quota Empty
1. Borrow office/Warung Wi-Fi (1 hour for meeting)
2. Friend's hotspot (treat coffee)
3. Buy emergency quota (Rp 2.000-5.000 for 500MB-1GB)

### Scenario 2: Urgent Call Needed, Credit Empty
1. WhatsApp call (if they use WA)
2. Borrow friend's credit (buy at warung, pay cash later)
   - Type: BIS [amount] send to [friend's number]
   - Example: BIS 5000 send to 08123456789
3. Public phone (terminal/station: Rp 100/sec)

### Scenario 3: Urgent Transfer, Internet Down
1. Ask friend/relative to transfer (pay cash later)
2. Use nearest ATM (inter-bank transfer available)
3. Buy emergency quota + internet banking

### Scenario 4: Out of Town, No Signal/Quota, Need Direction
1. Ask locals for directions
2. Look for street name boards
3. Use offline maps (if downloaded area beforehand)
4. Buy local SIM card (Tri local = Rp 10.000 for 2GB)

## Case Study: Dian (21 y.o., Barista)

**Problem**: Spent Rp 300.000/month on internet (Wi-Fi Rp 100.000 + quota Rp 100.000 + e-money Rp 100.000)

**Solution Applied**:
1. Stopped using kos Wi-Fi (more expensive than personal quota)
2. Switched to Smartfren (Rp 90.000 for 20GB)
3. Turned off auto-updates
4. Used "Data Saver" feature
5. Downloaded updates via free cafe Wi-Fi
6. Shared friend's MiFi modem (add Rp 15.000/month)

**Result**: Monthly internet cost = Rp 105.000 (saved Rp 195.000/month = Rp 585.000 in 3 months)

## Conclusion

Data saving isn't about being "poor" - it's about **discipline and planning**. UMR workers who manage quota wisely can save Rp 150.000-300.000/month. That's enough for:
- BPJS voluntary contribution
- 3-month emergency fund
- 2 private tutoring sessions for sibling
- Initial side hustle capital

### 5 Golden Rules:
1. **Choose operator by location**, not TV ads
2. **Turn OFF background data** for non-essential apps
3. **Use free Wi-Fi** for heavy activities (downloads/updates)
4. **Buy packages that match needs**, not the biggest
5. **Use helper apps** (Datally, YouTube Go) to monitor

**Special Message for UMR Workers**:
> "You're not stupid for using cheap quota or sharing Wi-Fi. You're smart for allocating money to important needs. Stupid people are those who pay expensive rates just because they're too lazy to manage quota or embarrassed in front of friends."
