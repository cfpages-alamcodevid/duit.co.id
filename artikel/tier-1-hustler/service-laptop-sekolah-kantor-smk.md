## Kenapa Service Laptop Sekolah & Kantor Kecil Masih Jaya?

Sekolah menengah atas (SMP/SMA/SMK) punya lab komputer 30-50 unit per sekolah. Tiap semester ada 5-10 unit yang kena virus, lemot, butuh cleaning atau upgrade SSD.

Kantor kecil (5-20 karyawan) di kota-kota satelit (Tangerang, Bekasi, Depok, Bogor) pakai laptop semua. Ndak punya in-house IT, kalau laptop rusak cari teknisi langganan.

Big service center (merek laptop) charge mahal: cleaning Rp 150 ribu, OS reinstall Rp 200 ribu, SSD upgrade Rp 300 ribu + harga SSD.

Kita bisa charge lebih murah: cleaning Rp 50-75 ribu, OS reinstall Rp 100 ribu, SSD upgrade Rp 150 ribu + harga SSD. Masih untung besar, pelanggan senang.

Math sederhana: Sekolah dengan 40 laptop, service 10 unit per bulan (cleaning + OS reinstall): 10 x (Rp 50rb + Rp 100rb) = Rp 1,5 juta per bulan per sekolah. Kalau dapat 3 sekolah langganan: Rp 4,5 juta per bulan. Lebih dari UMR Jakarta.

Saya baca banyak curhatan operator lab sekolah di Facebook group "Teknisi Laptop Indonesia". Mereka keluhkan service center resmi charge mahal, teknisi mandiri susah cari yang bisa dipercaya.

Data dari Kementerian Pendidikan 2025: Ada 55.000 sekolah menengah di Indonesia, rata-rata punya 30-40 unit komputer/laptop. Itu artinya 1,6 juta unit laptop butuh maintenance rutin.

Belum lagi kantor kelurahan, kecamatan, koperasi, dan UMKM yang pakai laptop untuk administrasi. Jumlahnya jauh lebih banyak.

## Detail Jasa: Laptop Cleaning (Debu & Thermal Paste)

Ini jasa paling sering diminta. Laptop sudah 1-2 tahun dipakai, kipas penuh debu, thermal paste kering, suhu laptop panas terus.

**Langkah-langkah:**

1. Bongkar casing laptop (simpan sekrup di wadah kecil, jangan sampai hilang)
2. Lepas kipas, bersihkan debu pakai kuas halus dan sikat gigi
3. Bersihkan heatsink dan ventilasi udara
4. Hapus thermal paste lama di prosesor pakai tissue + alcohol 70%
5. Oleskan thermal paste baru (Arctic MX-4) secukupnya (sebesar butir beras)
6. Pasang kembali kipas dan casing

Waktu pengerjaan: 30-45 menit per unit.

**Harga:** Rp 50.000 - Rp 75.000 per unit.

Kalau dapet order 10 unit sekaligus dari sekolah, charge Rp 50rb per unit. Total Rp 500rb. Selesai dalam 2 hari (5 unit/hari).

Saya hitung sendiri: 10 unit x 45 menit = 450 menit = 7,5 jam kerja. Dapat Rp 500rb. Rate per jam = Rp 66.666/jam. Jauh lebih tinggi dari UMR Jakarta Rp 4,9 juta/bulan = Rp 28.000/jam.

## Detail Jasa: SSD Upgrade (HDD Lambat ke SSD Kencang)

Ini jasa dengan margin paling gede. Laptop pakai HDD lama jadi lemot buka aplikasi, 5-10 menit boot Windows.

Ganti ke SSD: Boot cuma 10-15 detik, buka aplikasi instan.

**Langkah-langkah:**

1. Cek kapasitas HDD lama (500GB/1TB)
2. Beli SSD baru (256GB/512GB/1TB) - saran: selalu stok 2-3 buah
3. Clone OS dari HDD ke SSD pakai software (Macrium Reflect / EaseUS Todo Backup) lewat kabel SATA to USB
4. Ganti HDD lama dengan SSD di laptop
5. Test boot dan kecepatan

Waktu pengerjaan: 60-90 menit per unit (tergantung kapasitas HDD).

**Harga Jasa:** Rp 150.000 + harga SSD.

**Harga SSD (Market Price 2025-2026):**
- SSD 256GB: Rp 450.000 - Rp 500.000
- SSD 512GB: Rp 750.000 - Rp 800.000
- SSD 1TB: Rp 1.400.000 - Rp 1.500.000

Charge ke klien: Harga beli + markup 10-15%.

Contoh: Beli SSD 256GB Rp 500rb, jual ke klien Rp 575rb (markup Rp 75rb). Plus jasa Rp 150rb. Total tagihan Rp 725rb.

Untung per unit: Rp 75rb (markup SSD) + Rp 150rb (jasa) = Rp 225rb.

5 order SSD upgrade per bulan = Rp 1.125.000 untung murni.

## Detail Jasa: OS Reinstall & Data Backup

Laptop kena virus, Windows corrupt, atau mau install ulang biar bersih.

**Langkah-langkah:**

1. Backup data klien dulu (Document, Desktop, Picture, Video) ke eksternal HDD 1TB
2. Buat bootable Windows 10/11 pakai Rufus di flashdisk 32GB
3. Install ulang Windows (pilih partisi yang benar, jangan hapus data klien)
4. Install driver (download dari website resmi merk laptop: asus.com, dell.com, dll)
5. Aktifkan Windows (beli lisensi Rp 200rb atau tawarkan Linux Mint gratis)
6. Install aplikasi dasar (MS Office, Chrome, Firefox, Zoom, WhatsApp Desktop)
7. Restore data klien dari backup
8. Test semua berjalan normal

Waktu pengerjaan: 60-90 menit per unit.

**Harga:** Rp 100.000 per unit (sudah termasuk driver & aplikasi dasar).

**Data Backup:** Rp 50.000 per 100GB data.

Saya selalu sarankan ke klien: "Backup data dulu sebelum reinstall ya, biar aman." Mereka biasanya setuju, nambah Rp 50rb untuk backup.

Bulan lalu saya service 15 unit dari kantor kecil di Depok. Semua minta backup data + OS reinstall. Total: 15 x (Rp 100rb + Rp 50rb) = Rp 2,25 juta. Selesai dalam 3 hari.

## Trouble yang Mungkin Terjadi & Solusinya

**Laptop ndak mau nyala setelah dibongkar:**
- Cek kabel konektor baterai sudah terpasang sempurna
- Cek RAM sudah terpasang dengan klik yang benar
- Cek kabel fleksibel LCD sudah terhubung

**OS gagal install:**
- Pakai Windows installer original dari website Microsoft
- Cek laptop support UEFI atau Legacy BIOS
- Pastikan partisi SSD/HDD sudah di-format NTFS

**Data klien hilang:**
- SELALU backup dulu sebelum reinstall
- Double cek folder backup lengkap sebelum format
- Kalau masih hilang, pakai software recovery (Recuva) dari laptop lain

**Klien komplain laptop masih lemot:**
- Cek task manager, lihat aplikasi yang makan RAM
- Uninstall bloatware/aplikasi ndak perlu
- Sarankan upgrade RAM jika masih pakai 4GB

Saya sudah alami semua masalah ini. Pengalaman bikin kita makin jago. Jangan takut salah, yang penting jujur sama klien.
