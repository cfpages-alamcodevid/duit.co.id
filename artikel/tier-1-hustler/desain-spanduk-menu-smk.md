## Step-by-Step Desain Print-Ready Buat Pemula SMK

Gak perlu skill desain mahal, cukup tau cara drag-drop di Canva, lulusan SMK pasti udah diajarin.

### 1. Setup Tools (1 hari)
Daftar Canva Pro lewat https://www.canva.com/id_id/harga/, latian 5 template spanduk/menu gratis.

Pilih template kategori "Spanduk Warung Makan" atau "Menu Restoran", ganti teks, warna, sama foto sesuai permintaan warung.

Pastikan setting export: CMYK (bukan RGB), resolusi 300dpi, blee d 3mm di semua sisi.

> **⚠️ Tips:**
> Jangan kasih file sumber (Canva link atau PSD) ke pelanggan. Cuma kasih PDF print-ready, biar desain nggak diubah-ubah sembarangan.

### 2. Buat Portofolio (2 hari)
Desain 10 contoh gratis buat showcase di Instagram atau Facebook.

Contohnya: Spanduk "Warung Makan Buah Batu", menu "Nasi Padang Rp 15 ribu", katalog "Sembako Lengkap Harga Grosir".

Pasang watermark "Desain by [Nama Kamu]" di semua contoh, hapus watermark setelah pelanggan bayar lunas.

### 3. Format File Order
Buat Google Docs template invoice sederhana, isi: Nama Warung, Alamat, Jenis Desain, Harga, DP 50%, Sisa Bayar.

Link template gratis: https://klikpajak.id/blog/template-faktur-pajak-excel-gratis/

## Kesalahan Pemula & Cara Hindari

- **Kasih file sumber editable:** Jangan! Pelanggan bisa edit sendiri, nggak pesen lagi. Cuma kasih PDF print-ready.
- **Nggak minta DP:** Minta DP 50% dulu, kirim file setelah lunas. Invoice WA cukup, nggak perlu pake faktur resmi kalau belum punya NPWP.
- **Cetak sendiri:** Jangan dulu, partner sama percetakan lokal, dapet komisi 10% tanpa repot mikirin mesin cetak.
- **Harga terlalu murah:** Jangan jual Rp 25 ribu, warung bakal pikir kualitas rendah. Mulai dari Rp 75 ribu buat bangun review.

> **💡 Tips:**
> Kalau income udah capai Rp 4,8 juta per bulan, daftar NPWP pribadi gratis di djp.go.id lewat UU HPP No.7/2021 Pasal 4.
