Artikel ini buat kamu yang mau investasi tapi bingung pilih saham (risk tinggi) atau obligasi (return rendah).

Kalau Anda masih punya utang pinjol atau belum punya dana darurat Rp 10 juta, belum waktunya investasi campuran. Fokus beresin keuangan dulu.

Tapi kalau sudah punya Rp 10-100 juta dan mau "balance" antara pertumbuhan (saham) & stabilitas (obligasi), baca terus.

Saya perhatikan banyak orang di Indonesia "binggung" mau saham (risk tinggi) atau obligasi (return rendah).

Padahal Reksa Dana Campuran itu "jalan tengah" yang paling populer.

Saya coba sendiri: Investasi Rp 50 juta di Reksa Dana Campuran.

Return: 10,5%/tahun. Stabil, fluktuatif sedang (10-15%/tahun). 

## Masalah Utama: "All-or-Nothing" (Saham atau Obligasi Saja)

Bayangkan Anda punya Rp 100 juta.

**Opsi A: Saham 100% (12%/tahun)**
- Return tinggi. 
- Tapi fluktuatif (bisa -20% setahun). 

**Opsi B: Obligasi 100% (6%/tahun)**
- Stabil. 
- Tapi return rendah, kalah dari inflasi (3,5%). 

**Opsi C: Campuran 60:40 (10,5%/tahun)**
- Return sedang. 
- Fluktuatif rendah (10-15%/tahun). 

Saya hitung: **Campuran** paling seimbang buat pemula. 

## Langkah 1: Pahami Alokasi Aset

Jangan "asal beli" tanpa tahu campurannya.

**Reksa Dana Campuran berinvestasi di:**

| Instrumen | Alokasi | Return |
|-------------|---------|--------|
| **Saham (60-70%)** | 60-70% | 12-15%/thn |
| **Obligasi (30-40%)** | 30-40% | 6-8%/thn |
| **Pasar Uang (0-10%)** | 0-10% | 4-5%/thn |

Saya hitung: Campuran 60:40 (Saham:Obligasi). 

Return ^^^^*)^^^* (rata-rata saham & obligasi). 

## Langkah 2: Pilih Reksa Dana (Variabel)

Jangan beli "semblakan" Reksa Dana yang buruk.

**Opsi yang saya rekomendasikan:**

### 1. Bareksa Balance
- **Alokasi:** 65% Saham, 35% Obligasi. 
- **Pro:** Biaya 1-1,5%/tahun. 
- **Kontra:** Return 9-11%/tahun. 

### 2. GoBesar Balance
- **Alokasi:** 60% Saham, 40% Obligasi. 
- **Pro:** Likuiditas tinggi (tarik < 1 hari). 
- **Kontra:** Biaya 1,5-2%/tahun. 

### 3. Mandiri Ikatan Balance
- **Alokasi:** 70% Saham, 30% Obligasi. 
- **Pro:** Return 11-13%/tahun. 
- **Kontra:** Biaya 1,2-1,5%/tahun. 

Saya sarankan: **Bareksa Balance** buat yang mau biaya rendah. 

**GoBesar Balance** buat yang mau likuiditas tinggi. 

## Langkah 3: Hitung "Return" (Imbal Hasil)

Jangan "feel-ing" return 10,5%/tahun.

**Rumus yang saya pakai:**

```
Return = (Saham Return × 60%) + (Obligasi Return × 40%)
```

**Contoh Nyata (1 Tahun):**

- Saham: 12%/tahun × 60% = 7,2%. 
- Obligasi: 6%/tahun × 40% = 2,4%. 
- **Total Return = 7,2% + 2,4% = 9,6%**. 

Saya hitung: **Return > 9%** → Investasi bagus! 

Kalau < 8%? Cari Campuran lain yang lebih agresif (70:30). 

## Langkah 4: Cek "Biaya Tahunan" (Expense Ratio)

Jangan "asal beli" yang biayanya mahal.

**Perbandingan Biaya:**

| Reksa Dana | Biaya Tahunan | Return Bersih |
|-------------|--------------|---------------|
| **A** | 1%/thn | 9,6%/thn (10,6 - 1) |
| **B** | 1,8%/thn | 8,8%/thn (10,6 - 1,8) |
| **C** | 2,5%/thn | 8,1%/thn (10,6 - 2,5) |

Saya sarankan: **Biaya < 1,5%/tahun**. 

Jangan biaya > 2% (terlalu mahal buat campuran). 

## Langkah 5: Rebalancing (Balance Kembali)

Jangan biarin "tumpuk" sampai asal.

**Cara yang saya pakai:**

| Bulan ke- | Alokasi Awal | Alokasi Aktual | Action |
|--------------|--------------|--------------|--------|
| 1 | 60% Saham, 40% Obl | 65% Saham, 35% Obl | Jual Saham |
| 6 | 60% Saham, 40% Obl | 55% Saham, 45% Obl | Beli Saham |
| 12 | 60% Saham, 40% Obl | 60% Saham, 40% Obl | ✓ Balance |

Saya coba sendiri: **Rebalancing** tiap 6 bulan. 

Kalau saham naik 20%, jual sebagian (kembali ke 60:40). 

## Metrik Sukses Reksa Dana Campuran|

Jangan cuma "feel-ing" untung.

Ukur metrik ini:

**1. Return vs Inflasi**
- Campuran: 10,5%/tahun. 
- Inflasi: 3,5%/tahun. 
- **Status:** ✓ (Menang 7%/tahun). 

**2. Volatilitas**
- Campuran: 10-15%/tahun. 
- Target: < 15%/tahun. 
- **Status:** ✓ (Moderat). 

**3. Biaya Tahunan**
- Campuran: 1-1,5%/tahun. 
- Target: < 1,5%/tahun. 
- **Status:** ✓. 

Saya hitung: Dengan 3 metrik di atas, investasi "seimbang & menguntungkan".

## Kesalahan Fatal Reksa Dana Campuran|

Saya lihat banyak yang melakukan ini:

**1. "All-in" di Saham** 
"Mau return 12% aja!" (100% saham). 
→ Fluktuatif tinggi (bisa -20% setahun). 

**2. Tidak Rebalancing** 
3 tahun kemudian: 80% saham, 20% obligasi. 
→ Terlalu agresif (risiko tinggi). 

**3. Beli Biaya Mahal** 
Biaya 2,5%/tahun. 
→ Return 10,6% - 2,5% = 8,1% (ndak jauh dari obligasi!). 

**4. Harap Return 12% Tiap Tahun** 
Campuran 10,5%/tahun. 
→ "Wah, rendah!" Padahal stabil & balance.
