Artikel ini buat kamu yang mau pendapatan stabil 5-8% setahun tanpa ribet pantau pasar.

Kalau Anda masih punya utang pinjol atau belum punya dana darurat Rp 10 juta, belum waktunya investasi RDPT. Fokus beresin keuangan dulu.

Tapi kalau sudah punya Rp 10-100 juta dan mau pendapatan "stabil" (5-8%/tahun), baca terus.

Saya perhatikan banyak orang di Indonesia "binggung" antara deposito, sukuk, atau RDPT.

Padahal RDPT itu "jembatan" antara deposito (4%/tahun) dan saham (12%/tahun yang fluktuatif).

Saya coba sendiri: Investasi Rp 30 juta di RDPT.

Return: 6,8%/tahun. Stabil, ndak fluktuatif seperti saham. 

## Masalah Utama: "Cash Drag" & "Volatilitas"

Bayangkan Anda punya Rp 100 juta.

**Opsi A: Deposito (4%/tahun)**
- Aman, dijamin LPS. 
- Tapi return rendah, kalah dari inflasi (3,5%). 

**Opsi B: Saham (12%/tahun)**
- Return tinggi. 
- Tapi fluktuatif (bisa -20% setahun). 

**Opsi C: RDPT (6,8%/tahun)**
- Return sedang. 
- Stabil (volatilitas < 3%/tahun). 

Saya hitung: **RDPT** paling seimbang buat yang mau stabil. 

## Langkah 1: Pahami Cara Kerja RDPT

Jangan "asal beli" tanpa tahu cara kerjanya.

**RDPT berinvestasi di:**

| Instrumen | Jangka Waktu | Return |
|-------------|--------------|--------|
| **Obligasi (80-90%)** | 1-10 tahun | 6-8%/thn |
| **Sukuk (10-20%)** | 3-5 tahun | 6-7%/thn |
| **Pasar Uang (0-10%)** | 1-3 bulan | 4-5%/thn |

Saya hitung: RDPT 6,8%/tahun. 

Return > Deposito (4%), tapi < Saham (12%). Tapi **stabil!** 

## Langkah 2: Pilih Reksa Dana (Variabel)

Jangan beli "semblakan" Reksa Dana yang buruk.

**Opsi yang saya rekomendasikan:**

### 1. Bareksa Pendapatan Tetap
- **Pro:** Biaya 0,5-1%/tahun (murah). 
- **Kontra:** Likuiditas rendah (tarik 2-3 hari). 

### 2. GoBesar Pendapatan Tetap
- **Pro:** Likuiditas tinggi (tarik < 1 hari). 
- **Kontra:** Biaya 1-1,5%/tahun. 

### 3. Mandiri Ikatan Pendapatan Tetap
- **Pro:** Bisa "beli/tarik" tiap hari. 
- **Kontra:** Biaya 1-1,2%/tahun. 

Saya sarankan: **Bareksa** buat yang mau biaya rendah. 

**GoBesar** buat yang mau likuiditas tinggi. 

## Langkah 3: Hitung "Return" (Imbal Hasil)

Jangan "feel-ing" return 6,8%/tahun.

**Rumus yang saya pakai:**

```
Return = (Ending Value - Beginning Value) / Beginning Value × 100
```

**Contoh Nyata (1 Tahun):**

- Beginning: Rp 10 juta. 
- Ending: Rp 10,68 juta. 
- **Return = (10,68 - 10) / 10 × 100 = 6,8%**. 

Saya hitung: **Return > 6%** → Investasi bagus! 

Kalau < 5%? Cari RDPT lain yang lebih bagus. 

## Langkah 4: Cek "Biaya Tahunan" (Expense Ratio)

Jangan "asal beli" yang biayanya mahal.

**Perbandingan Biaya:**

| Reksa Dana | Biaya Tahunan | Return Bersih |
|-------------|--------------|---------------|
| **A** | 0,5%/thn | 6,3%/thn (6,8 - 0,5) |
| **B** | 1,2%/thn | 5,6%/thn (6,8 - 1,2) |
| **C** | 2%/thn | 4,8%/thn (6,8 - 2) |

Saya sarankan: **Biaya < 1%/tahun**. 

Jangan biaya > 1,5% (terlalu mahal buat RDPT). 

## Langkah 5: Strategi "Laddering" (Diversifikasi)

Jangan "tumpuk" semua di 1 jatuh tempo.

**Cara yang saya pakai:**

| Jangka Waktu | Jumlah | Instrumen | Alasan |
|--------------|---------|-------------|---------|
| Pendek (1-3 thn) | Rp 20jt | RDPT GoBesar | Likuiditas tinggi |
| Menengah (3-5 thn) | Rp 30jt | RDPT Bareksa | Biaya rendah |
| Panjang (5+ thn) | Rp 50jt | Sukuk Ritel | Return lebih tinggi |

Saya coba sendiri: **Laddering** 1-3-5 tahun. 

Dana pendek di RDPT likuid, dana panjang di Sukuk (return lebih tinggi). 

## Metrik Sukses RDPT|

Jangan cuma "feel-ing" untung.

Ukur metrik ini:

**1. Return vs Deposito**
- RDPT: 6,8%/tahun. 
- Deposito: 4%/tahun. 
- **Status:** ✓ (Lebih untung 2,8%/tahun). 

**2. Volatilitas**
- RDPT: < 3%/tahun. 
- Target: < 5%/tahun. 
- **Status:** ✓ (Stabil). 

**3. Biaya Tahunan**
- RDPT: 0,5-1%/tahun. 
- Target: < 1%/tahun. 
- **Status:** ✓. 

Saya hitung: Dengan 3 metrik di atas, investasi "aman, stabil, menguntungkan".

## Kesalahan Fatal RDPT|

Saya lihat banyak yang melakukan ini:

**1. "Parkir" Di Deposito** 
"Deposito aja" (4%/tahun). 
→ Kalah dari inflasi (rugi 3,5%/tahun!). 

**2. Tidak Cek Biaya Tahunan** 
Beli RDPT biaya 2%/tahun. 
→ Return 6,8% - 2% = 4,8% (ndak jauh dari deposito!). 

**3. "All-in" di 1 Platform** 
Rp 50 juta di Bareksa aja. 
→ Likuiditas rendah (tarik 2-3 hari). 
Harusnya: **Campur** Bareksa (biaya rendah) + GoBesar (likuid). 

**4. Harap Return 12% (seperti Saham)** 
RDPT cuma 6,8%/tahun. 
→ "Wah, rendah!" Padahal stabil & aman.
