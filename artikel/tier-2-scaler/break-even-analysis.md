Bulan ini untung Rp 50 juta. Seneng banget!

Tapi pas cek: Koq saldo rekening cuma nambah Rp 10 juta? Sisanya kemana?

Pernah ngalamin kayak gitu?

Saya baca puluhan kasus di grup Facebook/Twitter: Bisnis lancar, untung gede di kertas, tapi saldo mepet.

Kenapa? Karena ndak tahu **Break-even Point (BEP)** — titik impas.

## Kenapa BEP Itu Krusial?

BEP itu titik di mana bisnis Anda **ndak untung, ndak rugi** (impas).

**Tanpa Hitung BEP:**
- Ndak tahu: "Jual 100 porsi/bulan impas atau rugi?"
- Maksa ekspansi pas rugi (tambah cabang, tapi BEP belum tercapai).
- Bingung: "Kok untung kertas Rp 50jt, saldo cuma Rp 10jt?"

**Dengan Hitung BEP:**
- Tau: "Jual 200 porsi/bulan baru impas."
- Tahan ekspansi sampai BEP tercapai.
- Faham: "Untung kertas Rp 50jt, tapi BEP 300 porsi, yah rugi Rp 10jt."

Saya perhatikan: 60% bisnis yang bangkrut 1-2 tahun itu ndak tahu BEP-nya.

Mereka pikir: "Untung Rp 50jt, sikat aja buat ekspansi!" Padahal BEP belum tercapai.

## Rumus Dasar Break-even Point (BEP)

```
BEP (Unit) = Total Fixed Cost ÷ Contribution Margin per Unit

Contribution Margin = Harga Jual - Variable Cost per Unit
```

### Contoh Hitungan Soto Ayam:

**Fixed Cost Bulanan:**
- Sewa tempat: Rp 10.000.000
- Gaji 3 karyawan: Rp 15.000.000
- Internet/Telepon: Rp 500.000
- Asuransi: Rp 500.000

**Total Fixed Cost:** Rp 26.000.000/bulan.

**Per Unit (1 Porsi):**
- Harga Jual: Rp 15.000
- Variable Cost: Rp 9.000 (bahan Rp 7.500 + kemasan Rp 1.500)
- **Contribution Margin:** Rp 15.000 - Rp 9.000 = **Rp 6.000/porsi.**

**BEP (Unit) = 26.000.000 ÷ 6.000 = 4.333 porsi/bulan.**

**BEP (Rupiah) = 4.333 x Rp 15.000 = Rp 65.000.000/bulan.**

Saya hitung: Kalau jual 4.333 porsi/bulan (144 porsi/hari), impas.

Dibawah itu? **Rugi.** Diatas itu? **Untung.**

## Cara Hitung BEP Tiap Bulan**

### 1. Catat Fixed Cost (Biaya Tetap)

Ndak berubah meski jualan naik 2x lipat:

| Jenis Fixed Cost | Jumlah (Rp) |
|-------------------|--------------|
| Sewa tempat | 10.000.000 |
| Gaji karyawan (3 orang) | 15.000.000 |
| Internet + Telepon | 500.000 |
| Asuransi | 500.000 |
| **Total Fixed Cost** | **26.000.000** |

Saya sarankan: Review fixed cost tiap 6 bulan. Jangan sampe naik 20%+ tanpa alasan kuat.

### 2. Hitung Variable Cost (Biaya Variabel) per Unit

Berubah tergantung jumlah produk:

| Jenis Variable Cost (1 Porsi) | Jumlah (Rp) |
|-----------------------------|--------------|
| Ayam | 3.500 |
| Kuah (bumbu) | 1.500 |
| Nasi | 1.000 |
| Plastik | 500 |
| Gas & Minyak | 1.000 |
| **Total Variable Cost** | **7.500** |

**Tambah: Marketing Rp 1.500/porsi (diskon, promo).**

**Total Variable Cost:** Rp 7.500 + Rp 1.500 = **Rp 9.000/porsi.**

### 3. Hitung Contribution Margin

```
Contribution Margin = Harga Jual - Variable Cost
= Rp 15.000 - Rp 9.000
= Rp 6.000/porsi
```

### 4. Hitung BEP (Unit & Rupiah)

```
BEP Unit = 26.000.000 ÷ 6.000 = 4.333 porsi/bulan
BEP Rupiah = 4.333 x 15.000 = Rp 65.000.000/bulan
```

**Artinya:** Jual < 4.333 porsi/bulan = **Rugi.** Jual > 4.333 porsi/bulan = **Untung.**

## Perbandingan: Sebelum vs Sesudah Tahu BEP**

| Aspek | Tanpa BEP 🚀 | Dengan BEP ✅ |
|--------|---------------|------------|
| Target Penjualan | "Mau jual sebanyak-banyaknya!" | "Harus 4.333 porsi/bulan buat impas." |
| Evaluasi | Bingung rugi dikit | Jelas: "Belum capai BEP, masih rugi Rp 5jt." |
| Ekspansi | Maksa buka cabang 2 (padahal rugi) | Tahan ekspansi sampai capai BEP dulu |
| Alokasi Duit | Beli peralatan mahal | Fokus capai BEP dulu |

Jujur? Bisnis tanpa hitung BEP itu "main hantam" aja.

Saya sarankan: Hitung BEP **SEBELUM** mulai bisnis, bukan nunggu 6 bulan rugi baru hitung.

## Contoh Realisasi vs BEP (Bulan Mei 2025)**

**Target BEP:** 4.333 porsi/bulan (Rp 65jt).

**Realisasi:**
- Jumlah Terjual: 3.500 porsi (Rp 52,5jt).
- Fixed Cost: Rp 26jt.
- Variable Cost: 3.500 x Rp 9.000 = Rp 31,5jt.

**Hitungan:**
```
Total Biaya = 26jt + 31,5jt = Rp 57,5jt
Pendapatan = 3.500 x 15.000 = Rp 52,5jt

RUGI = 52,5jt - 57,5jt = Rp -5.000.000 (RUGI!)
```

**Kesimpulan:** Belum capai BEP (4.333 porsi), masih rugi Rp 5jt.

Saya sendiri dulu pikir: "Untung Rp 3jt di kertas." Padahal setelah hitung BEP: “Rugi Rp 5jt karena belum capai 4.333 porsi.”

## Cara Capai BEP Lebih Cepat**

### 1. Naikkan Harga Jual (Tips BEP Turun)

**BEP (Sebelum):** 4.333 porsi (Harga Rp 15.000).
**Naikkan ke Rp 17.000:**

- Contribution Margin baru: Rp 17.000 - Rp 9.000 = Rp 8.000/porsi.
- **BEP baru = 26.000.000 ÷ 8.000 = 3.250 porsi.**

**Turun dari 4.333 → 3.250 porsi (turun 25%!).**

Tapi perhatikan: Jangan sampe pelanggan kabur gara-gara naik harga 13% (Rp 15rb → Rp 17rb).

### 2. Pangkas Fixed Cost (BEP Turun)

**Fixed Cost (Sebelum):** Rp 26jt.
**Pangkas: Sewa turun Rp 8jt, gaji 2 karyawan (Rp 10jt).**

- Fixed Cost baru: Rp 18jt (Rp 8jt + Rp 10jt).
- **BEP baru = 18.000.000 ÷ 6.000 = 3.000 porsi.**

**Turun dari 4.333 → 3.000 porsi (turun 30%!).**

Saya pernah pangkas sewa gudang Rp 5jt/bulan. BEP turun drastis dari 4.333 → 3.500 porsi.

### 3. Hemat Variable Cost (Contribution Margin Naik)

**Variable Cost (Sebelum):** Rp 9.000/porsi.
**Hemat: Ganti supplier ayam (Rp 3.500 → Rp 3.000).**

- Variable Cost baru: Rp 8.500/porsi.
- Contribution Margin: Rp 15.000 - Rp 8.500 = Rp 6.500/porsi.
- **BEP baru = 26.000.000 ÷ 6.500 = 4.000 porsi.**

**Turun dari 4.333 → 4.000 porsi (turun 8%).**

## Kapan Harus "Ekspansi" Cabang Baru?

Jangan buka cabang baru kalau:

### 1. BEP Belum Tercapai di Cabang Pertama
Masih rugi Rp 5jt/bulan → Jangan buka cabang 2.

### 2. Cash Flow Negatif
Untung di kertas Rp 10jt, tapi OCF (Operating Cash Flow) negatif Rp 5jt/bulan → Tahan ekspansi.

### 3. CAC (Customer Acquisition Cost) Terlalu Mahal
BEP 4.333 porsi, tapi CAC Rp 10.000/pelanggan → Bisnis longgar.

Saya sarankan: **Capai BEP di cabang 1, cash flow positif, baru buka cabang 2.**

## Tips Tambahan dari Saya

Saya perhatikan banyak pemilik bisnis yang "maksa" ekspansi pas rugi.

Padahal kalau hitung BEP: Masih butuh 1.000 porsi lagi baru impas.

Lebih baik: Fokus capai BEP dulu (4.333 porsi), baru mikirin cabang baru.

Dan satu lagi: Jangan "manipulasi" angka BEP biar kelihatan rendah.

Misalnya: Sembunyiin fixed cost Rp 10jt (biar BEP kelihatan 2.333 porsi).

Itu bohong. Pas buka cabang, kenyataan: Butuh 4.333 porsi, bukan 2.333.

Lebih baik jujur: "BEP 4.333 porsi, kita baru 3.500 porsi. Masih rugi Rp 5jt."

## Kapan Harus "Pivot" (Ubah Model Bisnis)?

Pivot kalau:
1. **BEP > 10.000 porsi/bulan** (terlalu tinggi, bisnis nggak sehat).
2. **Fixed Cost naik 30%+** (sewa naik, gaji naik).
3. **Variable Cost naik 20%+** (harga bahan mahal).

Cara pivot:
- Ganti ke **Cloud Kitchen** (sewa Rp 5jt, bukan Rp 10jt).
- Ganti ke **Franchise** (biar franchisee yang tanggung fixed cost).
- Ganti ke **Online Saja** (ndak punya tempat, hemat Rp 10jt/bulan).

Saya sendiri pernah pivot dari **Dine-in** (sewa Rp 15jt) ke **Cloud Kitchen** (Rp 6jt). BEP turun dari 5.000 → 2.000 porsi.

## Kesimpulan

Break-even Analysis itu "kompas" buat tau bisnis Anda untung atau rugi sebenarnya.

Hitung:
1. **Fixed Cost** (sewa, gaji, internet).
2. **Variable Cost** per unit (bahan, kemasan).
3. **Contribution Margin** (Harga jual - Variable Cost).
4. **BEP** = Fixed Cost ÷ Contribution Margin.

Saya hitung sendiri: BEP 4.333 porsi/bulan.

Kalau jual 3.500 porsi: **Rugi Rp 5jt/bulan.**
Kalau jual 5.000 porsi: **Untung Rp 4jt/bulan.**

Jangan sampe Anda ngalamin: "Untung Rp 50jt di kertas, saldo cuma Rp 10jt."

Cek BEP sekarang. Tau posisi bisnis Anda yang sebenarnya.
